## Name

Scouring Charm

## Classification

Charm, despite the fact that Vanishing is taught in Transfiguration.

## Description

Vanish spills, stains, and other messes.

## Myths

* This charm is not a [banishing][] charm.  You would not use it to unpack a suit case.[^221215-1]
